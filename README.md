# SayOn Football Forecast App

A simple Premier League prediction game built for Ken.

## What You Need

- A computer with Python installed
- 5 minutes

## How To Run It (3 Steps)

### Step 1: Install Streamlit

Open Terminal (Mac) or Command Prompt (Windows) and type:

```bash
pip install streamlit
```

### Step 2: Run The App

Navigate to where you saved these files, then type:

```bash
streamlit run sayon_app.py
```

### Step 3: Use It

Your browser will open automatically with the app running.

---

## How To Deploy It Online (So Others Can Use It)

### Option A: Streamlit Community Cloud (FREE & EASIEST)

1. Go to https://share.streamlit.io/
2. Sign in with GitHub
3. Click "New app"
4. Upload these files to a GitHub repo
5. Point Streamlit to your repo
6. Done - you get a public URL

### Option B: Replit

1. Go to https://replit.com
2. Create new Repl
3. Upload these 2 files: sayon_app.py and requirements.txt
4. Click Run
5. Done - you get a public URL
